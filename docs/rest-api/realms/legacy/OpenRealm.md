## Open Realm
This endpoints makes the realm public and allows people to join.

| Host                        | Authentication |
| --------------------------- | -------------- |
| peoapi.minecraft.net        | ?              |

---

### Constructing the request
```
PUT /server/{id}/open
```

* **{id}** is the id of the realm *(from [List Realms](../list-realms/))*  

<br>

### Response
There is no response.
